# mnist分类系统
### 系统环境
    python3.8
    pytroch
    pillow
    pyqt5
### 主要功能
    1.图片的管理功能，图片上传、删除;
    3.图片分类功能实现，用户选择一副图片能识别图片数字内容;
    4.模型训练参数设定，系统给出模型训练的批次，轮次，学习率等参数设定界面。
### mnist介绍
    60000个训练数据 10000个测试数据
    0-9十个类
    28*28灰度图
### 模块介绍
    window文件夹为界面以及界面逻辑的代码
    test文件夹保存为模型和图片
    recognition文件夹为模型创建和模型的训练
### 注意事项
    第一次训练模型会下载mnist数据集
